MÉTODO querySelectorAll

A diferencia del método querySelector que retorna la referencia de un solo objeto HTML, el método querySelectorAll retorna una colección con todos los objetos HTML que cumplen el selector CSS pasado como parámetro. Es muy poderoso este método y nos da una gran facilidad para acceder a un conjunto de objetos que cumplen ciertas reglas.
