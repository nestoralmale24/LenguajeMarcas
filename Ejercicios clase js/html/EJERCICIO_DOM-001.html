INTRODUCCIÓN AL DOM. (DOCUMENT OBJECT MODEL).

El DOM es el Modelo de los Objetos que forman la página web (Document). En realidad se trata una interface que genera el navegador al leer nuestro html y que traduce las etiquetas en nodos (objetos), los cuales, son accesibles y manipulables desde javascript. Esto nos permite, entre otras cosas, modificar el estilo mediante javascript. Cosa que veremos a continuación.
